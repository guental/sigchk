# Signature Check
Filetypes: 7Z, AVI, EXE (BIN), GZ, JPG, MKV, MP4, PNG, RAR, XZ, ZIP, ZSTD.

# Usage
sigchk [FILE]...

# Requirements
`bash p7zip-full tar gzip xz-utils unrar zip unzip osslsigncode coreutils`

**Optional Packages**

`ffmpeg jpeginfo moreutils pngcheck zstd`

**Supported OS**

Guaranteed to work on GNU/Linux like Debian 10, Ubuntu 20.04 or higher
